# Stage 7 POE2 Route Plan

Last updated: 2026-07-05

## Goal

Validate a Path of Exile 2 guide/tool route without building a full build planner too early.

## Why A Lightweight Route

Path of Exile 2 is still changing during Early Access. A full planner has high maintenance cost:

- skill and passive changes
- item data changes
- balance changes
- economy volatility
- class and ascendancy changes

The low-risk route is to publish stable planning pages first and watch search/click behavior.

## Validation Metrics

Track:

- impressions for POE2 queries
- clicks to POE2 Lab
- guide CTA clicks
- planner roadmap clicks
- premium crossover clicks

## Do Next Only If There Is Signal

If POE2 pages get traffic:

1. Add a build intake form.
2. Add a simple build comparison helper.
3. Add shareable build notes.
4. Only then consider planner integrations.

Do not start with:

- full passive tree renderer
- item database
- DPS simulator
- economy tracker

## Source Refresh Checklist

Before expanding POE2 content, re-check:

- official Path of Exile 2 site
- Steam store page
- official news/patch posts
- current community build planner projects
