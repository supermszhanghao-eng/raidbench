POE2 starter build checklist

Use this checklist before copying a Path of Exile 2 build, especially while Early Access balance changes frequently.

See planner roadmap

Build sanity checks

  • Does the guide name the patch or update period?
  • Does it explain leveling skills separately from endgame skills?
  • Does it state required uniques, weapon type, and defensive layers?
  • Does it have a cheap fallback if trade prices move?
  • Does it explain boss damage windows and mobility needs?
