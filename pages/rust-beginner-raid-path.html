Rust beginner raid path

New raiders should avoid giant bases first. The goal is to pick narrow, affordable paths that teach scouting and cost control.

Plan a beginner raid

Starter path order

  1. Start with one or two sheet metal doors if the base looks active but small.
  2. Skip stone wall raids unless you have scouted a strong reason.
  3. Add one extra door to your sulfur plan because airlocks are common.
  4. Leave if the target becomes too expensive for the loot you expect.

Good beginner signals

  • Small footprint with visible activity.
  • No obvious compound or external tool cupboards.
  • Door path looks shorter than the wall path.
