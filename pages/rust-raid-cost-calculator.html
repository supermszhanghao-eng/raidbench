How to use it

  1. Pick the target: door, wall, or common base layer.
  2. Choose the raid method you expect to use.
  3. Add every target in the planned path, including backup doors and walls.
  4. Check the sulfur estimate, then add a buffer for missed splash and counters.

Sources and notes

RaidBench keeps conservative community estimates and should be checked after major Rust updates. Useful external references: Facepunch Rust Wiki, RustHelp, and community raid cost charts.
