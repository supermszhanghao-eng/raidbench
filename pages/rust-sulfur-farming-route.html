Rust sulfur farming route

Farm sulfur with a target number, not a vague feeling. Calculate the raid first, then set a route goal with a counter-raid buffer.

Find your sulfur target

Route planning formula

  1. Estimate the raid path in RaidBench.
  2. Add 15-25% sulfur buffer for missed splash, extra doors, and defense.
  3. Split farming into two or three bank runs instead of one greedy run.
  4. Stop farming when the raid target no longer looks worth the risk.

Example target

A plan that costs 9,440 sulfur should usually become a 11,000-12,000 sulfur farm target before you craft, especially if you expect counters.
