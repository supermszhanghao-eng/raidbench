Rust TC upkeep weekly sheet

Convert tool cupboard daily upkeep into weekly targets with a buffer before raiding or logging off.

Use calculator Get raid prep pack

Short answer

Multiply each daily tool cupboard resource by seven, then add a buffer before raids, long roams, or offline time. Track wood, stone, metal, and HQM separately.

ResourceDailyWeekly formulaBuffer idea
WoodTC valuedaily x 7Add one extra day if used.
StoneTC valuedaily x 7Add 10-20% for expansion.
Metal fragmentsTC valuedaily x 7Add buffer before upgrading doors.
HQMTC valuedaily x 7Track carefully; small shortages matter.

Checklist

  • Copy daily values from TC.
  • Multiply each material separately.
  • Add one emergency day.
  • Recheck after expanding.

Example

If the TC shows 2,500 stone and 800 metal per day, weekly targets are 17,500 stone and 5,600 metal before buffer.

Common mistakes

  • Only farming the largest material.
  • Forgetting HQM or metal after upgrades.
  • Not recalculating after honeycomb.
